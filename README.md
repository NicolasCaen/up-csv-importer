# UP CSV Importer

## Description
Créer, configurer et enregistrer des fichiers XML décrivant comment importer un CSV dans WordPress.

## Installation
1. Déposer le dossier dans `wp-content/plugins/`.
2. Activer le plugin dans l’administration WordPress.
